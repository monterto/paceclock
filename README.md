A pace clock app created using AI code tools for my own personal use.
![Screenshot](https://github.com/monterto/paceclock/blob/main/screenshots/paceclock-screenshot-01.png)

Features:
  - Swimmer's pace clock with four hands
  - Lap/Set counter that tracks times and splits
  - Optional rest timer
  - Optional ghost hand to track start/end times
  - Dark/light clock face
  - Installable as a PWA

Usage:
  - Tap the clock face to start the time
  - Tap the clock face again to record a split
  - Toggle the Rest button to automatically switch between working sets and rest
  - Tap Finish to stop the timer
  - Tap Reset to refresh the clock and start a new session

Installation:
  - Open https://monterto.github.io/paceclock/ in your browser
  - Open the overflow menu (three dots) in your web browser
  - Select "Add to Home screen"
  - Web browser will prompt for installation
